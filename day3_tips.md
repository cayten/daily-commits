# Day 3 – QA Best Practices 🌿

Today’s daily commit is about maintaining a consistent testing mindset 🧠

## 🧪 QA Best Practices

- Always read the requirements twice — once as a user, once as a tester.  
- Keep test cases **atomic**, **independent**, and **repeatable**.  
- Log meaningful bugs, not vague descriptions.  
- Automate smartly — not everything needs automation 😉  
- Keep learning: tools evolve, so should you. 🚀
